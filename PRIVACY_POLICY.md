# Privacy Policy - JS Timer

**Last updated: 2026-03-25**

## Data Collection

JS Timer does **not** collect, store, or transmit any personal data to external servers.

## Local Storage

All timer data (timer state, remaining time, running/paused status) is stored locally on your device using Chrome's `storage.local` API. This data never leaves your browser.

## Permissions

| Permission | Purpose |
|------------|---------|
| `storage` | Save timer state locally on your device |
| `alarms` | Trigger timer completion when countdown reaches zero |
| `notifications` | Display a desktop notification when the timer completes |
| `sidePanel` | Display the timer UI in the browser's side panel |

## Third-Party Services

JS Timer does not use any third-party services, analytics, or tracking.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/JongSeok-327/JS-Timer).
